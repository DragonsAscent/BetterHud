
# how to bring your head to the hud
create a yml file in your heads folder and enter it like this.

```
test_head:
  pixel: 4
```

![](https://i.imgur.com/4OA5xp8.png)


put it on the layout.

```
outline_head:
  heads:
    0:
      name: test_head
      x: 0
      y: 0
```

![](https://i.imgur.com/Ric4q9Y.png)
